Ontwikkelomgeving

Inleiding

Op deze pagina is te lezen welke software ik gebruik als ontwikkelomgeving. Ook beschrijf ik welke add-ons ik gebruik en waarom ik deze gebruik.

Software

Als softwareprogramma gebruik ik Visual Studio Code. Alhoewel ik voorheen Atom gebruikt als werkomgeving ben ik VS Code gaan gebruiken omdat deze een paar extra functionaliteiten heeft tegenover Atom. Zo heeft VS Code bijvoorbeeld een terminal ingebouwd waarbij ik commands kan doorvoeren en ik de console log kan lezen. Ook gebruik ik VS Code omdat dit ook door de meeste mensen binnen Fontys ICT wordt gebruikt, door studenten maar ook door docenten tijdens workshops. Verder gebruik ik VS Code voor de vele add-ons die ze in hun add-ons library hebben staan.

Extensies

Ook maak ik gebruik van verschillende extensies die mij helpen met de workflow. Dit is een lijst van extensies die ik momenteel gebruik:

  • Auto Rename Tag
  • Beautify
  • Color Highlight
  • Live Server
  • Rainbow Brackets
De meeste helpen om de code overzichtelijk en leesbaar te houden. Extensies zoals Auto Rename Tag en SCSS Everywhere gebruik ik om mijn workflow makkelijker te maken zijn omdat zij suggesties aangeven. Ik ben zelf ook blij met Live Server want door deze extensie heb ik veel meer tijd om te coderen aangezien ik niet steeds verschillende keren hoef te klikken voordat ik iets kan testen.

Git

Om structuur te houden tijdens het programmeren in groepsverband maken we gebruik van Github. Hier houden we al onze projecten bij zodat iedereen hier gebruik van kan maken.

Voor mijn onderdeel maak ik ook gebruik van Git. Dit helpt mij om overzicht te houden en iteraties te kunnen maken en laten zien. Ook komt het proces van werken hierin naar voren. Door hier te klikken ga je naar de Git-repository voor de interactieve video.
