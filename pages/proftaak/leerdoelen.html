UX SCO DEV PROF

Persoonlijke leerdoelen

Inleiding

Tijdens semester 4 was het de bedoeling om zelf een aantal leerdoelen op te stellen die betrekking hadden op bepaalde begrippen. Deze begrippen zijn oordeelsvorming, communicatie en leervaardigheden.

Oordeelsvorming

Voor oordeelsvorming heb ik het volgende leerdoel opgesteld: Ik wil ervoor zorgen dat producten en/of concepten kan bedenken op basis van goed onderzoek te doen.

In het begin van het semester ben ik begonnen met het vergaren van informatie. Deze informatie heb ik gekregen door onderzoek te doen. Ik heb veldonderzoek gedaan om informatie te krijgen over de kennis en mening van een specifieke doelgroep. Verder heb ik literatuuronderzoek toegepast om meer informatie te krijgen op onderwerpen zodat ik deze informatie op een later tijdstip kon gebruiken tijdens het project.

In een later stadium van het project heb ik ook onderzoek verricht door te controleren of mijn ontwerp ook wel gebruiksvriendelijk was. Dit onderzoek heb ik uitgevoerd om te controleren of mijn visie van het ontwerp overeenkwam met dat van de uiteindelijke gebruiker.

Door onderzoek te doen kan ik zeggen dat ik het leerdoel dat hierboven staat heb kunnen aantonen. Door eerst naar de juiste informatie te zoeken voorkom ik mogelijke fouten die in een project tot tijd en zelfs geldverlies kunnen leiden. En door het controleren via onderzoek weet ik beter dat het ontwerp dat ik gemaakt heb gebruiksvriendelijk is.

Communicatie

Voor communicatie heb ik het volgende leerdoel opgesteld: Ik wil ervoor zorgen dat mijn resultaten en conclusies onderbouwd kunnen worden waardoor ik andere kan overtuigen door op een gestructureerde manier te presenteren.

Door onderzoek te hebben verricht tijdens dit semester kan ik verantwoorden waarom ik bepaalde keuzes heb gemaakt. Daarnaast kan ik mijn keuzes ook baseren op bepaalde guidelines die ik gebruikt en gevolgd bij mijn ontwerpen. Als laatste kan ik de feedback die ik gekregen heb van de leerkrachten, groepsgenoten en de opdrachtgever gebruiken als onderbouwing.

Hierdoor kan ik zeggen dat ik het bovenstaande leerdoel heb aangetoond doordat ik argumenten kan leveren die mijn keuzes ondersteunen.

leervaardigheden

Voor leervaardigheden heb ik het volgende leerdoel opgesteld: Ik wil mijn kennis op het gebied van development vergroten zodat ik programmeren beter leer begrijpen en de mogelijkheden die het biedt.

Tijdens de laatste sprint heb ik gewerkt aan een interactieve video. Hiervoor heb ik geprogrammeerd in javascript. Ik heb hiervoor JSON voor de eerste keer gebruikt en heb het prototype uiteindelijk modulair gemaakt.

Verder heb ik de bijspijkerworkshops gevolgd omdat ik vind dat mijn programmeervaardigheden nog altijd onder het gemiddelde liggen van de student binnen media. Tijdens deze lessen heb ik geleerd om functies te maken om formuliervelden aan te vullen en hoe ik data uit externe bestanden kan halen en hoe de functies omgezet kunnen worden naar een module.

Door de bijspijkerworkshops te volgen en het interactieve video prototype te maken heb ik voortgang gemaakt met het vergroten van mijn development kennis. Alhoewel ik mijn kennis wel heb uitgebreid denk ik nog altijd dat mijn vaardigheden verbeterd kunnen worden. Er zijn namelijk nog veel elementen binnen het programmeren waar ik moeite mee heb of nog geen kennis over beschik.
